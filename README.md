# CMPT Robot 宣傳頁

這個資料夾是從 `/opt/CMPT_ROBOT/frontend/intro.html` 獨立複製出來的 GitHub Pages 宣傳版本。

## 檔案

- `index.html`：宣傳頁入口
- `assets/logo.avif`：頁面 logo / favicon
- `css/fonts.css`：本地字型設定
- `fonts/*.woff2`：Inter 字型檔

## 使用方式

把整個 `intro-github-promo` 資料夾內容推到 GitHub repo，即可用 GitHub Pages 發佈。

原始檔 `/opt/CMPT_ROBOT/frontend/intro.html` 未修改。
